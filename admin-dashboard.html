How to Use This Portal

1. Update a student's progress Go to All Students → click any row → edit the numbers (speeches, tournaments, hours, level) → click Save Progress. The student will see their updated stats immediately on their dashboard.
2. Add session notes Click any student → scroll to Session Notes in the popup → fill in date, session type, optional star rating, and your note text → click + Add Note. Students see these notes on their dashboard under "Session Notes".
3. Post announcements Go to Announcements → fill in title and message → click Post Announcement. All students will see it on their dashboard. You can delete old ones using the ✕ button.
4. Making yourself admin (first time) Register via register.html with your teacher email → go to Firebase Console → Firestore → students collection → find your document → change role from "student" to "admin" → save. Then log in via the Admin tab on the login page.
